About

This website is developed for an assignment for Software Development for the Web. The purpose of this webpage is to view time of different cities across the world. The time can either be viewed digitally or analog. The main components of this webpage are:

⇨  Single Page Application
⇨  Webserver on Node.js with the help of Express
⇨  Displaying customized world clocks
⇨  Retrieving/Updating settings by making server calls

For further insights or information about this project view the Github repository.
